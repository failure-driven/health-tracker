# Fitness Tracker

This project is for tracking fitness goals

# Installation

## Clone project

You can clone the project using

`gh repo clone failure-driven/health-tracker`

or 

`git clone https://github.com/failure-driven/health-tracker.git .`

## System Requirements

* Ruby version **ruby-3.0.0**

& Database **postgresql**

## Install Project Dependencies

Rails dependencies 

`bundle install`

JavaScript dependencies

`yarn install`

Initialise Database

`rails db:create`

Build project

`make build`

Migrate database

`rails db:migrate`

Seed database (seed file not updated yet)

`rails db:seed`

## Run file in the browser (development)

Start rails server

`rails s`

(localhost)[http://127.0.0.1:3000]
